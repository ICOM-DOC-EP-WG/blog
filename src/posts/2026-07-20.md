# **Step-by-Step Guide: Adding Entries to Zotero**

**For Online (Web) and Desktop** 
*Target: Add entries to the **[Exhibition and Performance Documentation](https://www.zotero.org/groups/6441975/exhibition_and_performance_documentation/collections/E562XEZF/collection)** public group, tag them, check duplicates, and validate.*

------

## **ONLINE (Web) Workflow**

*No installation required. Works in any browser.*

### **Step 1: Log In to Zotero Web**

1. Go to **[zotero.org](https://www.zotero.org/)** → Click **Log In** (top-right).
2. Enter your **Zotero account credentials**.
3. *(Optional)* Bookmark the **[group page](https://www.zotero.org/groups/6441975/exhibition_and_performance_documentation/collections/E562XEZF/collection)** for quick access.

------

### **Step 2: Join the Group (If Not Already a Member)**

1. Visit the **[group link](https://www.zotero.org/groups/6441975/exhibition_and_performance_documentation/collections/E562XEZF/collection)**.
2. Click **Join Group** (if public) 

------

### **Step 3: Add Entries (Online)**

Verify first that your entry is not already in the library.

#### **Option A: Manual Entry**

1. Navigate to the **"To Sort"** collection.
2. Click on the **magic wand** and enter the doi or ISBN 
3. If it doesn't work Click on **New Item** (green "+" button).
   Select the **item type** (e.g., Book, Journal Article, Web Page).
4. Fill in metadata (title, author, year, etc.).
5. Click **Save**.

#### **Option B: Import from Web (Browser Connector)**

1. Install the [Zotero Connector](https://www.zotero.org/download/) for your browser.
2. Navigate to the source (e.g., Google Scholar, journal page).
3. Click the **Zotero connector icon** in your browser toolbar.
4. Select the **group library** ("Exhibition and Performance Documentation") and the **"To Sort"** collection.
5. Click **Save**.

------

### **Step 4: Add Tags (Online)**

1. Select the entry(ies) in **"To Sort"**.
2. In the Tags field (right sidebar), type:
   - `exhibition`
   - `performance`
   - `documentation`
3. Press `Enter` after each tag.
4. (Bulk Tagging) :
   - Select multiple entries → Click **Add Tag** (tag icon) → Enter all 3 tags (separated by commas).

------

### **Step 5: Validate Entries (Online)**

1. Check metadata :
   - Ensure **title, author, year, publisher** are complete.
   - Add **DOI/ISBN/URL** in the **Identifier** or **URL** field.
2. Manually review :
   - Look for red error indicators or missing fields.
3. *(Limitation)* Online lacks "Clean Up Metadata." **Edit manually** for consistency.

------

### **Step 6: Move to Final Collection (Online)**

1. Select entries in **"To Sort"** → Click **Move To** (folder icon) → Choose the final collection.

------

## **💻 DESKTOP Workflow**

*Full-featured. Requires [Zotero desktop app](https://www.zotero.org/download/).*

### **Step 1: Log In to Zotero Desktop**

1. Open the **Zotero desktop app**.
2. Click **Edit** → **Preferences** → **Sync**.
3. Enter your **Zotero account credentials**.
4. Enable **Automatic sync** (recommended).
5. Click **Sync Now** to verify connection.

------

### **Step 2: Join the Group (Desktop)**

1. Click **File** → **Join Group**.
2. Enter the **group ID** (`6441975`) or search for **"Exhibition and Performance Documentation"**.
3. Click **Join** (or **Request to Join** if private).
4. The group will appear under **Group Libraries** in the left sidebar.

------

### **Step 3: Add Entries (Desktop)**

Verify first that your entry is not already in the library.

#### **Option A: Manual Entry**

1. Navigate to the **"To Sort"** collection.
2. Click on the **magic wand** and enter the doi or ISBN 
3. If it doesn't work Click on **New Item** (green "+" button).
   Select the **item type** (e.g., Book, Journal Article, Web Page).
4. Fill in metadata (title, author, year, etc.).
5. Click **Save**.

#### **Option B: Import from Web (Browser Connector)**

1. Install the [Zotero Connector](https://www.zotero.org/download/) if not already installed.
2. Navigate to the source (e.g., Google Scholar, PDF).
3. Click the **Zotero connector icon** in your browser.
4. In the popup, select:
   - **Library**: "Exhibition and Performance Documentation" (group).
   - **Collection**: "To Sort".
5. Click **Save**.

------

### **Step 5: Check for Duplicates (Desktop)**

1. Select the **"To Sort"** collection.
2. Click **Tools** → **Find Duplicates**.
3. Zotero will list potential duplicates. For each:
   - **Compare** titles/authors/years.
   - **Right-click** → **Merge Items** (if duplicate) or **Dismiss** (if unique).
4. *(Optional)* Run **Tools** → **Duplicate Items in Library** for a full group scan.

------

### **Step 6: Add Tags (Desktop)**

1. Select the entry(ies) in **"To Sort"**.
2. In the Tags field (right sidebar), type:
   - `exhibition`
   - `performance`
   - `documentation`
3. Press `Enter` after each tag.
4. (Bulk Tagging):
   - Select multiple entries → Right-click → **Add Tag** → Enter all 3 tags (separated by commas).

------

### **Step 7: Validate Entries (Desktop)**

1. Check metadata :
   - Ensure **title, author, year, publisher** are filled.
   - Add **DOI/ISBN/URL** in the **Identifier** or **URL** field.
2. Use "Clean Up Metadata":
   - Select entry → Right-click → **Clean Up Metadata** (fixes formatting, abbreviations, etc.).
3. Check for errors :
   - Red exclamation marks indicate issues. Hover for details.

------

### **Step 8: Move to Final Collection (Desktop)**

1. Drag validated entries from **"To Sort"** to the final collection.
